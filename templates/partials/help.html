Sign Up

If you are a new user you need to Sign Up first
If you are an old user ther the console will alert you by showing " UserName already Exists"
Then you will be redirected to the Sign-In Page

Sign In

Sign In if you an existing user
On successful Sign In you will be redirected to the Service Page

Service Page

On successful login you can use out services

  1. Select language
    2. Language in which you want to convert your audio to text
  2. Choose File
    3. Choose the video/audio file for which you want to generate transcripts
  3. Click Submit

Generate transcripts

Wait for few minutes for the video to get uploaded
Once the video is uploaded then on the right side you an find the transcripts are generated

  1. Plain Text
    2. Plain text for your transcripts are genrated which will help you to copy keywords
  2. Play Text
    3. text with anchor tags are genrated which will help you to navigate the video to your desired keyword
  3. Edit Text
    4. If the words in the transcripts are not generated accurately the you can edit the text

Add Keywords

When certains keywords are repeating in the text then you can add keywords to find out their counts
Keyword counts are displayed in the count table
You can also delete the keywords simultaneously

Search Keywords

Search for keywords in a particular video or in entire videos' list

Analytics page

Your Videos, keywords, keyword counts are displayed in a graphical representation in the analytics page
